# Movie X-Ray Generator

Proyecto desarrollado en Python que identifica automáticamente los actores que aparecen en una película utilizando DeepFace y TMDb.

## Características

- Descarga automáticamente el reparto desde TMDb.
- Construye una base de embeddings faciales.
- Analiza un vídeo.
- Genera un JSON con la línea temporal de actores.
- Incluye un reproductor en PyQt6 + VLC que muestra un panel X-Ray sincronizado con el vídeo.

## Tecnologías

- Python
- DeepFace
- TensorFlow
- OpenCV
- PyQt6
- VLC
- TMDb API
